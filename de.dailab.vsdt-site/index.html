Introduction

Visual Service Design Tool (VSDT) is a Tool for the graphical development of services. As a notation the Business Process Modelling Notation is used. The Business Process Diagrams can be exported to executable BPEL code. A preliminary transformation to JIAC V is included and a more elaborate mapping to agents is currently under development.

Installation

For more information please refer to the Manual.

To install the latest Version of the Visual Service Design Tool...

  1. Get and install Eclipse 3.4 Ganymede
  2. Click Help -> Software Updates
  3. Install GMF using the Ganymede Discovery Site (including dependencies)
  4. Add a new Update Site with URL=http://repositories.dai-labor.de/sites/vsdt/
  5. Select the newly created site and select the VSDT features to install.

Download

Alternatively to installing the VSDT using the Update Site, you can also download the complete package as an archive. Be sure you have the required GMF plugins (Version 2.0 for VSDT 1.0.x, and Version 2.1 for VSDT 1.1.x and 1.2.x), then copy the contained 'features' and 'plugins' directories into the directory of your local Eclipse installation, merging them with the equally named existing folders.
We strongly recommend installing the VSDT using this Update Site. However, downloading the VSDT can be useful for installing it on a device that is not connected to the internet.

Usage

For more information please refer to the Manual.

To create a new diagram...

  1. Open the Easy Service Creation Perspective: Window -> Open Perspective -> Other ...
  2. Create a new project: File -> New -> Project
  3. Right-click the project, select New -> BPMN Diagram
  4. Name both the model and the diagram file (preferably with the same prefix)
To export BPMN to BPEL...
  1. Validate your BPMN Diagram (in the Diagram Editor: Diagram -> Validate)
  2. Right-click a model file (extension .bpmn) in the Navigator view
  3. Select Export... -> BPMN Export -> BPMN to BPEL
  4. Follow the Wizard and eventually press finish
